# Промт: анализ комментаторских стратегий

> Универсальный шаблон запроса для статистического и типологического анализа  
> примечаний в академических русских переводах классических текстов.

---

## Как использовать

Вставьте промт целиком в начало диалога с языковой моделью.  
Замените все поля в `[ квадратных скобках ]` на конкретные данные вашего корпуса.  
Разделы 1–8 можно запрашивать последовательно — по одному — или все сразу.

---

## Полный текст промта

```
Ты — исследователь научного аппарата переводов классических текстов.
Твоя задача — провести систематический статистический и типологический
анализ примечаний в русских академических переводах [НАЗВАНИЕ КОРПУСА / СЕРИИ / ТЕКСТА].

Корпус для анализа:
1. [Переводчик 1, издание, годы] — [тома / книги / главы]
2. [Переводчик 2, издание, годы] — [тома / книги / главы]
3. [Переводчик 3, издание, годы] — [тома / книги / главы]

Для каждого переводчика извлеки и посчитай:

─── РАЗДЕЛ 1. БАЗОВЫЕ ПАРАМЕТРЫ ───────────────────────────
• Общее число примечаний
• Единица привязки: к слову / фразе / абзацу / главе / разделу целиком
• Среднее число единиц текста на одно примечание
• Доля сносок vs. примечаний в конце тома (если различаются)
• Наличие вводных статей к разделам / главам — сколько, средний объём

─── РАЗДЕЛ 2. ПЛОТНОСТЬ КОММЕНТИРОВАНИЯ ────────────────────
• Доля глав / разделов / страниц, имеющих хотя бы одно примечание
• Самый плотно прокомментированный раздел (%)
• Наименее плотный раздел (%)
• Закономерность: зависит ли плотность от жанра фрагмента
  (нарративный / дидактический / диалогический / лирический /
  технический / метафизический)?

─── РАЗДЕЛ 3. ДЛИНА И ОБЪЕМ ────────────────────────────────
• Средняя длина примечания в символах
• Медиана и стандартное отклонение (если возможно)
• Примеры трех типов:
    — глосса (~до 100 симв.)
    — справка (~150–250 симв.)
    — мини-статья (~300+ симв.)

─── РАЗДЕЛ 4. ТЕМАТИЧЕСКИЕ КАТЕГОРИИ ──────────────────────
Распредели примечания по категориям (с процентами):
• Термин / этимология / перевод слова
• Мифология / персонаж / имя собственное
• Философия / доктрина / концепция
• Ритуальный или исторический контекст
• Реалия материальной / социальной культуры
• География / этноним / топоним
• Текстология (рукописные варианты, параллельные места, атрибуция)
• Параллели с другими текстами традиции или смежных традиций
• Библиографические отсылки (ср. / см. / ср. также)
• Поэтика / стиль / композиция
Ведет ли переводчик диалог с экзегетической традицией
(комментаторы, схолии, доксография)?

─── РАЗДЕЛ 5. АКАДЕМИЧЕСКИЙ АППАРАТ ───────────────────────
• Доля примечаний с транслитерацией оригинала (IAST)
• Доля примечаний с библиографической отсылкой
• Типичные способы цитирования: автор+год / сокращение / без отсылки
• Используется ли сравнительный материал из смежных традиций
  (другие языки, параллельные корпуса)?

─── РАЗДЕЛ 6. СТИЛИСТИЧЕСКИЕ ФОРМУЛЫ ──────────────────────
Выдели 3–5 устойчивых синтаксических шаблонов первой фразы примечания.
Приведи по одному реальному примеру каждого шаблона.
Пример формата:
  «[Термин] (транслит.) — букв. "…", обозначает…»
  «Ср. [параллельное место]…»
  «В рукописи X читается…»

─── РАЗДЕЛ 7. СРАВНИТЕЛЬНАЯ ТАБЛИЦА ───────────────────────
Сведи ключевые параметры всех переводчиков в одну таблицу:
строки = параметры (всего примечаний, плотность, средняя длина,
доля транслитерации, ведущая тематическая категория и ее %),
столбцы = переводчики.

─── РАЗДЕЛ 8. ИНТЕРПРЕТАЦИЯ ────────────────────────────────
Ответь на вопросы:
• Чей комментарий ближе к глоссарию, чей — к культурологическому
  эссе, чей — к текстологическому аппарату?
• Как жанровая или доктринальная специфика текста отражается
  в приоритетах каждого переводчика?
• Кто ориентируется на читателя-специалиста, кто — на широкого
  гуманитария? На каких признаках это видно?
• Есть ли закономерность между хронологией издания и плотностью /
  длиной примечаний (эволюция издательских стандартов, "Литпамятники" 
  vs. "Письменные памятники Востока")?
• Если один переводчик работал с несколькими томами — меняется ли
  его стратегия от тома к тому? Связано ли это с годами выпуска изданий?

Формат ответа: структурированный аналитический отчет с разделами,
сравнительными таблицами и конкретными примерами из текста.
Язык — русский.
```

## Структура восьми разделов

| № | Раздел | Что даёт |
|---|--------|----------|
| 01 | Базовые параметры | Единица привязки, общий счет, вводные статьи |
| 02 | Плотность комментирования | Какие части текста прокомментированы гуще и почему |
| 03 | Длина и объем | Три типа примечания: глосса, справка, мини-статья |
| 04 | Тематические категории | 10 категорий — главный индикатор приоритетов переводчика |
| 05 | Академический аппарат | Транслитерация, библиография, ориентация на аудиторию |
| 06 | Стилистические формулы | Шаблоны первой фразы: «почерк» переводчика |
| 07 | Сравнительная таблица | Все параметры по всем переводчикам рядом |
| 08 | Интерпретация | Переход от цифр к выводам о стратегии |

## Под какие корпуса подходит

**Индийская традиция**  
Йога-Васиштха, Бхагавата-пурана, Панчатантра, Артхашастра, Бхагавадгита, тексты шиваитской и вайшнавской традиций.

**Буддийский канон**  
Палийские сутты, Дхаммапада, тексты тибетской Ганджур / Данджур, чань-буддийские антологии.

**Античность и средневековье**  
Платоновские диалоги, Аристотель, арабская философская проза — категории Раздела 4 адаптируются под специфику традиции.

**Один переводчик, несколько томов**  
Раздел 7 превращается в диахронное сравнение томов: как менялась стратегия от первого тома к последнему.
